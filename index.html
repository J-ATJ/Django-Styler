Django Styler App Preview

This project is a web app created with Django, which is a modern Python framework for web development.

The app allows users to upload an image and apply one of the several filters from the Stylerpy library created by me with OpenCV in Python. OpenCV is an open-source library used for image processing and computer vision.

Filters include paint, comic, sketch, pencil, cartoon, among others. When the form is sent, the application returns the original image and the image with the filter applied.

The app is responsive and has a simple design using bootstrap 5.
